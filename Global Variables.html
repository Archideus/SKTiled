Global Variables

SKTiled provides several global variables for working with or getting information about the framework:

Variable Description
SKTILED_VERSION Returns the SKTiled framework version.
SKTILED_CONTENT_SCALE Returns content scale for the current device.
SKTILED_LOGGING_LEVEL SKTiled logging level.
SKTILED_USE_SPRITEKIT_ACTIONS Use SpriteKit actions for tile animations.
  • SKTILED_USE_SPRITEKIT_ACTIONS
